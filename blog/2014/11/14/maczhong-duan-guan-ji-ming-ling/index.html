Mac终端关机命令

Nov 14th, 2014 2:00 pm

立即关机

sudo halt

或者

sudo shutdown -h now

10分钟后关机

sudo shutdown -h +10

晚上8点关机

sudo shutdown -h 20:00

立即重启

sudo reboot

或者

sudo shutdown -r now
